# Intelligent Cloud Infrastructure Review with GenAI Assistance

## Project Overview
This project focuses on reviewing cloud infrastructure configurations to identify
security risks, inefficiencies, and optimization opportunities using GenAI-assisted analysis.

## Problem Statement
IT leadership requires better visibility into cloud usage, security risks, and
operational inefficiencies. Full automation is out of scope, so GenAI is used only
to assist in analysis and recommendations.
